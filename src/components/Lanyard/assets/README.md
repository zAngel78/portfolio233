# Lanyard Assets

Necesitas descargar estos 2 archivos del repositorio de React Bits:

## Archivos necesarios:
1. **card.glb** - Modelo 3D de la tarjeta
2. **lanyard.png** - Textura del cordón

## Dónde descargarlos:
Ve a: https://github.com/davidhaz/react-bits/tree/main/src/assets/lanyard

O busca el repositorio de React Bits en GitHub y navega a `src/assets/lanyard/`

## Instrucciones:
1. Descarga `card.glb`
2. Descarga `lanyard.png`
3. Coloca ambos archivos en esta carpeta (`src/components/Lanyard/assets/`)

## Personalización (opcional):
- Puedes editar `card.glb` en https://modelviewer.dev/editor/ para cambiar la textura de la tarjeta
- Puedes editar `lanyard.png` en cualquier editor de imágenes para personalizar el diseño del cordón
